# Cloud Kitchen Premium

A premium startup-style Cloud Kitchen landing page built with React, Vite, Tailwind CSS, Framer Motion, and Lucide icons.

## Features

- Full-screen hero with animated CTA
- Glassmorphism, floating gradients, premium cards
- Popular dishes, offers, reviews, gallery, FAQ, contact
- Smooth page animations with Framer Motion
- Theme toggle, scroll progress indicator, back-to-top button
- Responsive layout and high-end aesthetic

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
